## 실행방법
```
1. env 파일에 NYT_API_KEY, NYT_API_SECRET, NYT_API_URL 변수를 추가해주셔야 합니다.
2. yarn install
3. yarn run dev 
```

<br />

## 스택
- 리액트, 웹팩으로 환경 세팅하였습니다.
- 리덕스툴킷을 사용하여 상태관리를 하였습니다.

<br />

## 참고 부탁드립니다.
짧은 간격으로 api 호출을 하지 않아도 종종 429 에러가 (Too Many Request) 발생하고, apiKey가 차단당할 우려가 있습니다.
